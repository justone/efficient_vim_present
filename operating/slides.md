# Modifiers

* i - inner
* a - around
* t - till
* f - find
* / - search

# Text Objects

* w, W - word
* b, B - block
* t - tag
* ', " - quoted
* (), {}, [], &lt;&gt; - block
* s, p - sentence/paragraph
